# mean-rBox

Recipe box MEAN stack application.

* MongoDB
* Express
* AngularJS (with [reStart-angular](https://github.com/kmaida/reStart-angular))
* Node
* Token-based authentication (with Oauth via [Satellizer](https://github.com/sahat/satellizer))
* FontAwesome (CDN)
* Bootstrap 3 (local)
* Gulp
